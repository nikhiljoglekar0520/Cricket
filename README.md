# Cricket Game

A desktop Cricket Game developed using **Python** and **Pygame** featuring an interactive graphical interface, batting and bowling gameplay, player animations, and match result screens.

---

## Features

- 🏏 Interactive cricket gameplay
- 🎮 Graphical user interface using Pygame
- 👤 Batting and bowling animations
- 🖼️ Custom game assets and sprites
- 📊 Match result screen
- ⚡ Smooth game controls

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Cricket-Game.git
```

Move into the project directory:

```bash
cd Cricket-Game
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment.

### Windows

```bash
.\.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Game

```bash
python CRICKET.py
```
